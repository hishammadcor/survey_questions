Place .wav and .lab files in this folder.

You will also need a manifest.json file generated by the helper script.
Format (array of objects):
[
  {
    "id": "unique-id-1",              // optional
    "audio": "alpha_data/file1.wav",   // required, relative path used by the web app
    "label": "Sentence from file1.lab",// required, the text content of the .lab file
    "filename": "file1.wav"            // optional, used for display
  }
]
